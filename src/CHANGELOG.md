# 版本更新日志

## 3.1.3

- 修复了 `JAB/JetBrains IDE 支持`
- 重构了延迟隐藏的处理逻辑

## 3.1.2

- 修复了 `开机自启动` 和 `以管理员权限运行` 的相关问题
- 其他的优化和修复

## 3.1.0

**新功能**

- 符号方案的特殊偏移量现在支持标题级匹配，允许为单个应用的不同窗口设置特殊偏移量
- 允许符号方案的文本类型符号内容设置为空，表示在该状态下不显示符号
- 允许浮窗方案的文本设置为空，表示在该状态下不显示浮窗

**性能优化**

- 通过预解析窗口规则并使用 Map 映射，大幅提升了窗口规则的匹配性能

**问题修复**

- 修复了符号方案中偏移量未生效的问题
- 修复了浮窗方案和符号方案中透明度失效的问题
- 修复了从 3.0.0 中数据未完整迁移的问题
- 修复了部分配置更改后未及时生效的问题

**其他改进**

- 为配置修改添加防抖，以解决配置未正常写入的问题
- 补充了更多配置项的合法性校验
- 优化了文件下载逻辑和下载界面
- 更新并统一了控件渲染相关方法
- 优化了部分菜单布局
- 其他优化与修复

## 3.0.0

- 新增 `浮窗方案` ，支持自定义样式、位置和过渡动画，为状态提示带来更优雅的视觉体验
- 新增 `颜色选择器`，所有颜色配置项均可通过可视化调色盘设置，无需手动输入色值
- 符号方案中的方块符号升级为 `形状符号`，新增圆角样式，选择「小圆角」后将变为圆形
- 配置菜单全面支持 `i18n` 国际化
- 新增对日文和韩文输入法的状态识别支持
- 新增对美式键盘与中文输入法混用场景下的多键盘布局状态识别支持
- 为鼠标方案支持设置为空，表示在该状态下还原系统默认鼠标样式
- 浮窗方案和所有菜单添加了流畅的过渡动画
- 窗口新增圆角和深色模式标题栏支持
- 为所有配置项添加合法性校验，配置异常时自动回退到默认值
- 优化了窗口规则的查找性能
- 新增鼠标穿透，移除符号悬停隐藏，改为固定实时显示
- 重构了配置文件、数据目录和菜单布局，统一代码风格与命名规范
- AHK 运行环境升级至 v2.0.26
- 版本号从时间回归到 X.Y.Z
- 其他的优化和修复
